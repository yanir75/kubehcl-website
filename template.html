template Command

The template command converts HCL configuration files into YAML or JSON manifests. This allows you to preview the Kubernetes resources that will be created in your environment before applying them.

← Back to all commands

Usage

kubehcl template [flags]

Flags

  • -h, --help — Help for template.
  • --kind string — Prints the template in YAML or JSON format (default "yaml").
  • --var strings — Set a specific variable's value; must contain an equals sign.
  • --var-file string — Vars filename to load values into variables (default "kubehcl.tfvars").
