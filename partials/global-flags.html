Global Flags

  • --burst-limit int — Client-side default throttling limit (default 100)
  • --content-cache string — Path to the directory containing cached content (e.g. charts)
  • --debug — Enable verbose output
  • --kube-apiserver string — Address and port of the Kubernetes API server
  • --kube-as-group stringArray — Group(s) to impersonate for the operation (can be repeated)
  • --kube-as-user string — Username to impersonate for the operation
  • --kube-ca-file string — Certificate authority file for the Kubernetes API server connection
  • --kube-context string — Name of the kubeconfig context to use
  • --kube-insecure-skip-tls-verify — Skip certificate validity check (insecure)
  • --kube-tls-server-name string — Server name to use for TLS validation
  • --kube-token string — Bearer token used for authentication
  • --kubeconfig string — Path to the kubeconfig file
  • -n, --namespace string — Namespace scope for this request
  • --no-color — Disable color in command output
  • --qps float32 — Queries per second used when communicating with the Kubernetes API
  • --registry-config string — Path to the registry config file
  • --repository-cache string — Path to the cached repository indexes
  • --repository-config string — Path to the repositories configuration file
  • --timeout int — Timeout for each resource creation (default 100)
