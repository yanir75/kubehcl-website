Usage

Define your Kubernetes resources in HCL and let KubeHCL generate YAML automatically:


kube_resource "foo" {
  apiVersion = "v1"
  kind       = "Namespace"
  metadata = {
    name = "foo"
    labels = {
      name = "bar"
    }
  }
}
