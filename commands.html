Available Commands

  • create Create an example folder for your initial release.
  • fmt Format all files in the folder.
  • install Create or update resources.
  • list List all modules.
  • plan Show changes that will be made.
  • pull Pull a module from the repository.
  • push Push a module to the OCI registry (only supports OCI).
  • repo Manage modules in repositories using subcommands.
  • template Print the resources that will be created in YAML or JSON format.
  • uninstall Uninstall all resources managed by KubeHCL.
  • version Show the current version of the tool.
