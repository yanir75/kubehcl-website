pull Command

The pull command downloads a module from a repository using the provided tag and name. The module must be a .tar.gz file, and KubeHCL will automatically create a folder for it locally.

← Back to all commands

Usage

kubehcl pull [REPO NAME] [TAG/NAME] [flags]

Flags

  • -h, --help — Help for pull.
  • --version string — Specify the module version to pull (usable only in repositories).
