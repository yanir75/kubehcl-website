install Command

The install command creates or updates existing resources managed by KubeHCL. It automatically searches for files ending with .hcl and applies them to your cluster.

← Back to all commands

Usage

kubehcl install [name] [folder] [flags]

Flags

  • --create-namespace — Automatically create the namespace if it doesn’t exist.
  • -h, --help — Help for install.
  • --var strings — Set specific variable values (must include an "=" sign).
  • --var-file string — Variables filename to load values (default "kubehcl.tfvars").
