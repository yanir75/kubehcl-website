locals block

The locals block defines local values that can be used throughout your configuration. It contains attribute names and their assigned values.

Local values can be referenced in configuration files using the syntax local.<name>, for example: local.foo.

Variables defined in variable blocks can also be used within locals.

Example


variable "namespace" {
  default = "default"
}

locals {
  app_name  = "my-app"
  app_label = "${var.namespace}-label"
}
