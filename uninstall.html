uninstall Command

The uninstall command destroys existing resources managed by KubeHCL. It searches for .hcl files and removes the corresponding Kubernetes resources.

← Back to all commands

Usage

kubehcl uninstall [name] [folder] [flags]

Flags

  • -h, --help — Help for uninstall.
  • --no-color — Disable colored output.
  • --var strings — Set a specific variable’s value; must contain an equals sign.
  • --var-file string — Vars filename to load values into variables (default "kubehcl.tfvars").
