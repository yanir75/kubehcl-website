repo Command

The repo command provides the option to add, list, or remove repositories containing KubeHCL modules.

← Back to all commands

Usage

kubehcl repo [flags]
kubehcl repo [command]

Available Commands

  • add — Add a repository of modules.
  • list — List all added repositories.
  • remove — Remove a repository of modules.

Flags

  • -h, --help — Help for repo.

Additional Help

Use kubehcl repo [command] --help for more information about a specific subcommand.
