plan Command

The plan command shows the differences between existing resources managed by KubeHCL and the desired configuration. It automatically searches for files ending with .hcl and generates a change plan.

← Back to all commands

Usage

kubehcl plan [name] [folder] [flags]

Flags

  • -h, --help — Help for plan.
  • --var strings — Set a specific variable’s value (must include an equals sign).
  • --var-file string — Vars filename to load variable values (default kubehcl.tfvars).
