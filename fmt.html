fmt Command

The fmt command formats all files in the specified folder. It helps maintain a consistent structure and style across your configuration files.

← Back to all commands

Usage

kubehcl fmt [folder] [flags]

Flags

  • -h, --help — Help for fmt.
  • --no-color — Disable colored output.
  • --recursive — Formats all the folders and subfolders
