Installation

You can install KubeHCL on macOS, Linux, or Windows. Choose your platform below for specific instructions.

üß© macOS Installation

Install via Homebrew using the official tap:


# Add the KubeHCL Homebrew tap
brew tap yanir75/homebrew-apps https://github.com/yanir75/homebrew-apps

# Install KubeHCL
brew install kubehcl

üêß Linux Installation

Download the appropriate pre-built binary for Linux from the KubeHCL Releases page. 


# Visit the releases page:
https://github.com/yanir75/kubehcl/releases

# 1. Download the matching "kubehcl_<version>_linux_<processor>.tar.gz" (amd,arm etc) asset
# 2. tar xvzf kubehcl_<version>_linux_<processor>.tar.gz
# 3. Make it executable
chmod +x kubehcl_<version>_linux_<processor>/kubehcl

# 4. Move it into your PATH for global use
sudo mv kubehcl_<version>_linux_<processor>/kubehcl /usr/local/bin/kubehcl

# 5. Verify installation
kubehcl --version

ü™ü Windows Installation

Download the Windows binary from the KubeHCL Releases page. 


# Visit the releases page:
https://github.com/yanir75/kubehcl/releases

# 1. Download the matching "kubehcl_<version>_windows_<processor>.tar.gz" asset
# 2. Extract the file and rename it to kubehcl.exe
# 3. Add the folder containing kubehcl.exe to your system PATH
# 4. Open PowerShell or Command Prompt and verify:
kubehcl --version
