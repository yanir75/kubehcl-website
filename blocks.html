Blocks

The following blocks are available in KubeHCL configuration files:

  • variable Defines input variables for configurations.
  • locals Defines reusable local values within the configuration.
  • backend_storage Specifies where the state is stored (e.g. kube_secret, stateless).
  • default_annotations Sets default annotations for Kubernetes resources.
  • kube_resource Defines Kubernetes resources managed by kubehcl.
  • module Includes and configures other kubehcl modules.
